# Квартик

## Краткое описание

- **Вид**: Мечерог (тип монстра).
- **Роль**: Вожак стаи.
- **Встреча**: Леград встречает его при разведке туннелей Чёрной горы для создания ловушки Зверей.

## Описание

- Обладает умом, достаточным для организации стаи.
- Возможно, имеет повышенную духовную силу по сравнению с обычными мечерогами.
- Имеет склонность к обману (использует туннели как ловушку).
