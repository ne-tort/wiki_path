# Тренировочная площадка

## Краткое описание

- Установлены полки с гирями разного веса.
- Ряды камней для подъёма.
- Утоптанная земля.
- Используется для развития силы и выносливости учеников.
- Рядом находится площадка для драк (Ринг).
