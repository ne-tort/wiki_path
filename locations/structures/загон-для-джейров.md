# Загон для джейров

## Краткое описание

Открытый загон в деревне, где содержатся джейры — домашние животные, источник мяса и шкур.

## Описание

- В деревне, рядом с жилыми домами
- Возле дома Котила (пастуха)

- [Джейр](/creatures/beasts/джейр) — многочисленные особи
- **Вожак джейров** — особо крупный и сильный джейр, лидер стада

- Содержание джейров перед убоем
- Леград использует для тайных тренировок печатей ограничения (Глава 17)
- Место, где Леград изучает возможности своих печатей
