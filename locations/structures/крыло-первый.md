# Крыло «Первый»

## Краткое описание

Часть общего здания Школы. Свой большой двор, коридоры, классы, ринг. Ученики класса «Первый» живут и учатся здесь. Комната, куда привели Леграда и Зимиона, узкая, без окон, освещалась магическим шаром. Скамьи по стенам.

## Описание

- **Классовая комната** (та, что узкая)
- **Ринг** (очерченная площадка)
- **Зона отдыха** (навес со столами)

- Учитель: [Иглис](/characters/secondary/иглис)
- Ученики: [Леград](/characters/main/леград), [Зимион](/characters/secondary/зимион), [Виликор](/characters/secondary/виликор), другие таланты.
