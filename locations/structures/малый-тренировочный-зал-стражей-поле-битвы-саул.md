# Малый Тренировочный Зал Стражей (Поле Битвы Саул)

## Краткое описание

Руины Малого Тренировочного Зала Стражей на [Поле Битвы Саул](/locations/structures/поле-битвы-саул) — крепость Стражей, стоявших на защите Империи Сынов Неба. Пала около трёхсот лет назад (вместе с городом при нападении Альянса Тысячи Сект). На карте Поля Битвы отмечена как «пустое» пятно, без наград для идущих.

## Описание

**Описание руин**: разрушенные ворота (изломанные кинжалы шипов ворот), колючая ограда, разросшаяся по площади и руинам; нет башен со шпилями, полированный камень стен покрыт следами времени и проломами; самый большой пролом — на месте ворот. Нет флагов с алым гербом Стражей (феникс с шестью изогнутыми перьями хвоста). Внутри — следы древней схватки (вспоротые стены, ямы в брусчатке, обломки брони и мечей), пещера с бугристым полом (от формации испытания остался щебень), коридор к хранилищу наград (полки пусты), зал для эрия и высоких рангов (пусто; вероятно, прежние Стражи превратили город в Поле Битвы). Проход в глубине перегорожен формацией Древних: надпись «Только для Стражей», герб Феникса. Шэн может провести через неё гостя. За формацией — внутренняя пещера со светящимися стенами, десятками проходов (пещерки для сна со светильниками из ядра Зверя, ложа со шкурами), круг для медитаций, пещера с ручьём и озерцом; в нишах под формацией — одежда Стражей (чёрная со вставками стихий). У развалин Зала в гл. 8 находился флаг «Миллиона» (источник призраков), уничтоженный Леградом.
