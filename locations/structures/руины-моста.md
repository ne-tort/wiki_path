# Руины моста

## Краткое описание

- Место тренировок Леграда (глава 5-7).
- Леград использует подводные ступени для тренировки закалки в воде.
- В руинах много щелей, тайников, каменного крошеву — Леград находит там каменный шип и позже кости Рауха.
- Верхняя площадка на "Пальце" (одинокая руина) — гнездо Рауха.

## Описание

- Никто не живёт, но место посещают охотники-подростки для добычи камней.
- Леград использует как убежище.
