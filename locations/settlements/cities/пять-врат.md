# Пять Врат

## Краткое описание

Город на юге земель Саул, перед [Пепельными Равнинами](/locations/regions/пепельные-равнины). По вести оттуда Зотар узнал о следе семьи Леграда: старик-старший со стихией дерева приехал на двух телегах с попутчиками (два старика, двое мужчин, две женщины, парень, девушка, девочка). Северная Стена — следующий город после Пяти Врат по направлению к Пепельным Равнинам (оттуда, по легенде, прибыл отряд «Даргела»).
