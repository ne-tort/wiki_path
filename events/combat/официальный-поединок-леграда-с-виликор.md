# Официальный поединок Леграда с Виликор

## Краткое описание

Первый учебный поединок (7 дней с начала занятий): Шамор объявляет пары, Леград просит Виликор.

Первый учебный поединок (7 дней с начала занятий): Шамор объявляет пары, Леград просит Виликор.

Шамор ставит пары; победитель получает 20 баллов. Леград просит сразиться с Виликор — Шамор соглашается, предупреждает Виликор: не калечить (ни ударов между ног, в горло, глаза, ничего не ломать). Леград насыщает тело силой (без средоточия, прорыв на 11-ю звезду), держится, защищается, бьёт в ответ — у Виликор идёт кровь из носа. Виликор усиливает натиск, Леград стоит, контратакует (её отшвыривает к краю ринга). Шамор останавливает бой: «Довольно! Увиденного достаточно. Каждому по десять баллов.» Виликор: «Я могу победить!» — Шамор: «Минус два балла за пререкания.»
