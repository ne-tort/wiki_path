# Отрубание Фатией головы Урката и распространение слухов о Жуках

## Краткое описание

В [Глава 3](/books/11-император-земли-итрэй/глава-3) у выхода из пещеры на [Хребете Трав](/locations/regions/хребет-трав) [Фатия](/characters/secondary/фатия) отрубает голову Урката (убитого в Главе 2 сектанта [секты Каменные Жуки](/organizations/sects/секта-каменные-жуки)) как доказательство для круга старейшин — он пытался изнасиловать и убить дочь главы секты. [Леград](/characters/main/леград) даёт состав для высушивания добычи (ватажников), голова обрабатывается, черты остаются узнаваемыми. По пути вниз Фатия при встречах с другими сектантами объявляет: она — дочь главы [секты Сапфировые Тритоны](/organizations/sects/секта-сапфировые-тритоны), ищет встречи с Жуками; один из них поднял на неё руку, они «сильно должны». Слухи разносятся по Хребту, чтобы к моменту разбирательства на круге старейшин вина Жуков была общеизвестна.
