# Разговор Леграда с Даритом об обучалке

## Краткое описание

Во дворе класса Дарит замечает, что Леград достал «обучалку» — ею могут пользоваться только Воины; делает вывод, что Леград прорвался. Объясняет: артефакт Древних, дети Древних бегали с ними на учёбу; для активации нужны средоточие и передача силы в камень (волной в руку, выпустить из ладони, коснуться). Леград пробует — камень открывает красный глазок, над рукой возникает полупрозрачная картина с символами. Дарит говорит: содержимое только для владельца; при создании мастера вкладывали проверочный текст (например, правила поведения за столом), важное давно переписано на бумагу; сейчас это пустышка для коллекционеров.
