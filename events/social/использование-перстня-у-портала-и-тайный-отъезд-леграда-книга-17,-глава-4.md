# Использование перстня у портала и тайный отъезд Леграда (Книга 17, Глава 4)

## Краткое описание

Леград приходит к площадке портала под восстановленным навесом; стража и идущий, управляющий формацией, на месте. Приказывает «оставить нас» — стража уходит. С драконьим перстнем (голова внутрь ладони) видит десятки плывущих строк — Седой их не видит. Выбирает «Пять Ветров»; система сообщает: массив вне списка для уровня и места, затем «заблокирован для прибытия». Леград расширяет список только для себя, выбирает привычную обходную точку, ставит в очередь: после перехода троих закрыть портал и скрыть следы. Для проверки вызывает Рутгоша мыслью и задаёт вопрос — Рутгош не замечает; приказывает перстню не оставлять следов — проверка успешна. Приказывает идущему открыть портал, объявляет старейшинам срок возвращения и старшего — Бахара; с Пересмешником и Седым шагает в портал.
