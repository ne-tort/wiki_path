# Опора

## Краткое описание

Техника «прирастания» к земле: Возвышающийся направляет энергию из средоточия по меридиану к узлу в ноге, рисует силой круг под ногой с одним символом (символ единства с землёй). На один-два вдоха, пока печать сияет, идущего невозможно сдвинуть с места — можно полноценно использовать физическую силу без потери опоры.
