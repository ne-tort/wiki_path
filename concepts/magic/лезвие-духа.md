# Лезвие Духа

## Краткое описание

- **Тип**: Дистанционная атака.
- **Действие**: Создаёт полупрозрачное лезвие из энергии Неба, которое можно метнуть.
- **Дальность**: До нескольких шагов.
- **Урон**: Соответствует силе удара мечом (при хорошем освоении).
- **Контроль**: Требует концентрации для наведения.

## Описание

- Открытый узел (часто в ладони или пальце).
- Печать для формирования лезвия.
- Основные этапы: создание, удержание, запуск, управление траекторией.
- Можно создавать несколько лезвий одновременно при высоком уровне.
