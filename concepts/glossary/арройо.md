# Арройо

## Краткое описание

- **Уровень**: Пятизвёздочный (высокий для Нулевого круга).
- **Инфраструктура**: 
  - Дома из дерева (редкость в Пустоши).
  - Широкие улицы, выложенные камнем.
  - Мастерские, торговые лавки.
- **Население**: Жители имеют более высокий статус.
- **Законы**: Для доступа к некоторым возможностям требуется пройти экзамен (Римило хотел вернуться и пройти экзамен).

## Описание

- [Римило](/characters/secondary/римило) — бывший кузнец.
- Родители [Римило](/characters/secondary/римило).
- [Варо](/characters/secondary/варо) — брат Римило.
- [Эри](/characters/secondary/эри) — родилась в трущобах, вышла замуж за Римило, была отвергнута родителями мужа.

- Дом Римило: большой, стол из дерева, кадки с цветами.
- Мастерская кузнеца.
