# Караваны

## Краткое описание

- Основной способ перевозки товаров и людей между поселениями.
- Состоит из повозок, впряжённых животных (быки, джейры?).
- Торговцы используют караваны для доставки товаров и покупки местных продуктов.
- Позволяет переносить письма, как мама пыталась отправить письмо Варо.

## Описание

- **Торговля**: Завоз редких товаров (зенья, инструменты), вывоз местной добычи.
- **Коммуникация**: Письма, информация.
- **Транспорт**: Для желающих переехать (семья Римило планировала устроиться в караван).
- **Опасность**: Караваны подвержены нападениям монстров, разбойников.

- «Караванщик» — участник каравана, проводник.
- «Караван из Арройо» — большой караван.
- «Караван из двух торговцев» — маленький.
