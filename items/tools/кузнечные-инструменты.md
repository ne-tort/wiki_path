# Кузнечные инструменты

## Краткое описание

Набор инструментов, оставленных отцом [Леграда](/characters/main/леград) — [Римило](/characters/secondary/римило). Содержит клещи, ручник, гладилку, раскатку, гвоздильню, зубило, подсечку, обжимку, подбойки, пробойники и молот. Хранятся в полированном сундучке из дерева мопани и кожи буйвола.

## Описание

- **Клещи** — для удержания раскалённого металла.
- **Ручник** — для подъёма.
- **Гладилка** — выравнивание.
- **Раскатка** — прокатка.
- **Гвоздильня** — для производства гвоздей.
- **Зубило** — резка.
- **Подсечка** — отрезание.
- **Обжимка** — формовка.
- **Подбойки** — поддержка.
- **Пробойники** — сверление, сделанные из стали, которой нет в Нулевом (артефакт).
- **Молот** — основной инструмент, но, по мнению [Кардо](/characters/secondary/кардо), самая дорогая вещь. Леград считает, что пробойники ценнее.

- **Качество**: Инструменты высокого класса, некоторые (пробойники) сделаны из стали, недоступной в Нулевом.
- **История**: Передались от деда отца, возможно, ещё древнее.
- **Значение**: Для Леграда — связь с отцом, символ его наследия и потенциальный старт для новой жизни.
- **Магия**: Пока не проявляют особых свойств, но их сталь может быть уникальной.
